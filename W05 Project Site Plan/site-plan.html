Sun Paws Shelter — Site Plan

WDD 231 — Web Frontend Development I

1. Site Name

Sun Paws Shelter

This name evokes warmth, shelter, and hope for rescued animals. It is memorable and appropriate for an adoption-focused site.

2. Site Purpose

The website provides information about the shelter, lists adoptable pets, and allows visitors to submit adoption requests. It also offers resources on responsible pet ownership.

3. Scenarios

  • How can I view the pets that are currently available for adoption?
  • Where can I submit an adoption request online?

4. Color Scheme

#D98324 — Primary accent (headings, buttons, highlights)

#FFF8E6 — Background and soft areas

#4A6B57 — Secondary text and accents

5. Typography

Poppins — Used for headings and body text to ensure consistency and modern appearance.

6. Wireframe

Mobile

Header • Hero • Mission • List • Footer

Desktop

Header • Hero (image + text) • Two-column content • Footer

7. Testing

  • HTML validation: W3C Markup Validator
  • Accessibility: WAVE
  • Performance & SEO: Lighthouse
  • Color contrast: WebAIM Contrast Checker
